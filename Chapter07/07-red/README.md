# Chapter 07

## Prerequisites

The following software is required to run the sample:

- Git
- Node.js
- NPM

## Running

Clone the repository:

```sh
git clone https://github.com/ArtOfMicrofrontends/07-red.git
```

Go to the repository's directory and run NPM install:

```sh
npm install
```

Now start the application:

```sh
npm start
```

## Steps

Follow these steps to implement the same from scratch.

1. Initialize the repository

```sh
npm init -y
```

2. Install the dependencies

```sh
npm i ejs express
```

3. Add code for the microfrontend

4. Add HTML fragments into `views`

5. Add the required assets (e.g., CSS stylesheets) into `public`

6. Run server
